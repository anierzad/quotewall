# Quote Wall

Application to allow the submission of funny quotes from the office which can then be displayed on a display within the office.

## Technologies

### API

	* Node
	* Express
	* MongoDB

### Client

	* Node
	* Express
	* Angular

## Features

### Core

 	* Add users
 	* Add quotes
 	* View quotes
 	* Quote visualisation

### Future
	
	* Conversation quotes
		* e.g. Me: blah blah, You: blah blah, Me: blah blah blah!
	* Voting
	* Most viewed
	* Most voted
	* User leaderboard
